# Новые страницы из NewPage/

Эта папка содержит React-компоненты, созданные из HTML-файлов из папки `NewPage/`.

## Структура

- `TailwindWrapper.jsx` - Компонент-обертка для подключения Tailwind CSS через CDN
- `NewHomePage.jsx` - Главная страница (Главная.html)
- `NewServicesPage.jsx` - Страница услуг (Услуги.html)
- `NewBlogPage.jsx` - Блог (Блог.html) - TODO
- `NewCasesPage.jsx` - Кейсы (Кейсы.html) - TODO
- `NewAboutPage.jsx` - О нас (О нас.html) - TODO
- `NewTeamPage.jsx` - Наша команда (Наша команда.html) - TODO
- `NewVacanciesPage.jsx` - Вакансии (Вакансии.html) - TODO
- `NewServiceTemplatePage.jsx` - Шаблон страницы услуг (Шаблон страницы услуг.html) - TODO

## Роуты

Все новые страницы доступны под префиксом `/new/*`:
- `/new` - Главная
- `/new/services` - Услуги
- `/new/blog` - Блог (TODO)
- `/new/cases` - Кейсы (TODO)
- `/new/about` - О нас (TODO)
- `/new/team` - Наша команда (TODO)
- `/new/vacancies` - Вакансии (TODO)
- `/new/service-template` - Шаблон страницы услуг (TODO)

## Важно

Все компоненты должны строго сохранять HTML-структуру, классы, data-атрибуты и ID из оригинальных HTML-файлов.

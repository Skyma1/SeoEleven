# Генерация фавиконов

Для создания всех необходимых форматов фавикона из `favicon.svg` используйте один из следующих сервисов:

## Рекомендуемые сервисы

1. **RealFaviconGenerator** (https://realfavicongenerator.net/)
   - Загрузите `favicon.svg`
   - Настройте параметры для разных платформ
   - Скачайте все файлы

2. **Favicon.io** (https://favicon.io/)
   - Конвертирует SVG в различные форматы
   - Генерирует все необходимые размеры

3. **Favicon Generator** (https://www.favicon-generator.org/)
   - Поддерживает SVG
   - Генерирует все форматы

## Необходимые файлы

После генерации разместите следующие файлы в папке `public/favicon/`:

- ✅ `favicon.svg` - уже создан
- ⬜ `favicon.ico` - классический фавикон (16x16, 32x32, 48x48)
- ⬜ `apple-touch-icon.png` - для iOS (180x180px)
- ⬜ `favicon-16x16.png` - 16x16px
- ⬜ `favicon-32x32.png` - 32x32px
- ⬜ `android-chrome-192x192.png` - для Android (192x192px)
- ⬜ `android-chrome-512x512.png` - для Android (512x512px)
- ⬜ `site.webmanifest` - манифест для PWA (опционально)

## Примечание

Файл `favicon.svg` уже создан и будет использоваться современными браузерами. Остальные форматы нужны для совместимости со старыми браузерами и мобильными платформами.

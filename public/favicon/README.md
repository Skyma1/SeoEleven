# Фавикон

Эта папка предназначена для хранения фавиконов сайта.

## Необходимые файлы

Разместите следующие файлы фавикона:

### Основные форматы

- **favicon.ico** - Классический фавикон (16x16, 32x32, 48x48 пикселей)
- **favicon.svg** - Векторный фавикон (предпочтительно для современных браузеров)
- **apple-touch-icon.png** - Иконка для iOS (180x180px)
- **android-chrome-192x192.png** - Иконка для Android (192x192px)
- **android-chrome-512x512.png** - Иконка для Android (512x512px)

### Дополнительные форматы (опционально)

- **favicon-16x16.png** - 16x16px
- **favicon-32x32.png** - 32x32px
- **site.webmanifest** - Манифест для PWA

## Требования

- **favicon.ico**: должен содержать несколько размеров (16x16, 32x32, 48x48)
- **favicon.svg**: векторный формат, масштабируется без потери качества
- **apple-touch-icon.png**: 180x180px, без скругления (iOS добавит автоматически)
- **android-chrome**: квадратные иконки 192x192 и 512x512px

## Использование

Ссылки на фавиконы уже добавлены в `public/index.html`. После размещения файлов они будут автоматически подхвачены браузером.

## Генерация фавиконов

Можно использовать онлайн-сервисы для генерации всех необходимых размеров:
- https://realfavicongenerator.net/
- https://favicon.io/
- https://www.favicon-generator.org/

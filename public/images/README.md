# Логотипы и изображения

Эта папка предназначена для хранения логотипов и других изображений сайта.

## Структура файлов

### Логотипы

Разместите следующие файлы логотипа:

- **logo.png** - Основной логотип (рекомендуемый размер: 200x50px или больше)
- **logo.svg** - Векторный логотип (предпочтительно для лучшего качества)
- **logo-dark.png** - Логотип для темной темы (если отличается)
- **logo-light.png** - Логотип для светлой темы (если отличается)

### Использование в коде

Логотипы можно использовать следующим образом:

```jsx
// В React компонентах
<img src="/images/logo.svg" alt="SeoEleven" />

// Или импортировать
import logo from '/images/logo.svg';
```

### Требования к логотипу

- Формат: SVG (предпочтительно), PNG с прозрачным фоном
- Минимальный размер: 200x50px для PNG
- Фон: прозрачный
- Цвета: должны хорошо смотреться на светлом и темном фоне

## Примечания

- Файлы в папке `public/` доступны напрямую по URL: `/images/filename.ext`
- При изменении логотипа обновите также ссылки в `public/index.html` (Schema.org разметка)
